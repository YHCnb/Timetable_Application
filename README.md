# Timetable_Application
![image](https://github.com/YHCnb/Timetable_Application/assets/112797916/d6eee131-53db-47f7-9383-e57d557af1e5)
一款简单的课程表Android软件，轻松实现课程管理

有什么优势？ 
- 清新简洁的外观
- 简单直观的交互体验，避免繁杂的操作
- 支持高度自由的定义，打造独属于你的时间规划表！
- 实现BIT官网、ICS文件一键导入课表，1s效率神器
